# Forgotten Games
A website that documents fixes and ways to play old computer games from the windows 98/xp era.

### Disclaimer

This repository and website are intended solely for the preservation and educational enjoyment of older computer games. All content, including guides, patches, and resources, is provided without the intent to infringe on copyrights, circumvent copy protection, or distribute proprietary game files. Users are responsible for ensuring that they own or legally acquire any software mentioned or used.

The purpose of this project is to document and provide technical solutions, fixes, and historical context for older games, helping enthusiasts experience them on modern systems in a way that respects original ownership and intellectual property rights.
